# GoLedger UI 设计规范 — 轻盈暖色系

> 设计关键词：**轻盈、留白、线性、温暖、克制**
>
> 面向大众的记账工具，UI 应该像纸质账本一样安静舒适，不抢注意力，让数字本身成为主角。

---

## 一、色彩体系

### 主色

| 角色 | 色值 | 说明 |
|------|------|------|
| Primary | `#F28C5E` 暖杏橘 | 主操作按钮、选中态、FAB，饱和度适中不刺眼 |
| Primary Light | `#FDEEE5` 极浅杏 | 选中背景、Tag 底色、输入框聚焦底色 |

只用一个主色，保持视觉统一，不做渐变。

### 语义色

| 角色 | 色值 | 说明 |
|------|------|------|
| 支出 | `#E87461` 柔珊瑚 | 支出金额、支出图表 |
| 收入 | `#66B89A` 薄荷绿 | 收入金额、收入图表 |
| 警告 | `#F2A85E` 暖黄 | 余额不足等提示 |
| 成功 | `#66B89A` 复用收入色 | Toast 成功提示 |

支出和收入只靠颜色区分，不加多余装饰。

### 中性色

| 角色 | 色值 | 说明 |
|------|------|------|
| 背景 | `#FAFAF8` 暖灰白 | 页面底色，比纯白温暖一点点 |
| 卡片 | `#FFFFFF` | 卡片表面 |
| 分割线 | `#F0EDE8` | 极淡，几乎看不见 |
| 主文字 | `#2D2A26` 炭棕 | 标题、金额 |
| 副文字 | `#9E9890` 灰棕 | 时间、备注、标签 |
| 占位文字 | `#C8C3BC` | 输入框 placeholder |

### 暗色模式

| 角色 | 色值 |
|------|------|
| 背景 | `#1A1816` |
| 卡片 | `#242220` |
| 分割线 | `#2E2C28` |
| 主文字 | `#EDE8E0` |
| 副文字 | `#8A847C` |
| Primary | `#F2A070`（亮度微调，保证对比度） |

## 二、排版

```
金额大字:    28sp / 700 weight / DIN Alternate 或系统等宽
页面标题:    18sp / 600 weight
卡片标题:    15sp / 500 weight
正文:        14sp / 400 weight
辅助文字:    12sp / 400 weight / 灰棕色
```

- 中文用系统默认字体
- 金额数字用等宽字体，让小数点对齐
- 行高统一 1.5 倍

---

## 三、圆角与间距

| 组件 | 圆角 | 说明 |
|------|------|------|
| 卡片 | 12dp | 不要太圆，保持干净 |
| 按钮 | 10dp | |
| 输入框 | 8dp | |
| 品类图标容器 | 圆形 | 36dp 圆 |
| 底部弹出面板 | 顶部 16dp | |

**间距节奏**：以 8dp 为基础单位，常用 8 / 12 / 16 / 24 / 32。

**阴影**：几乎不用。卡片靠 1dp 极淡描边 `#F0EDE8` 或微弱阴影 `elevation: 0.5` 区分层级。轻盈感的关键是去掉重阴影。

---

## 四、图标风格

- **线性图标**，线宽 1.5dp，圆角端点
- 品类图标：36dp 圆形容器，背景 `#FDEEE5`（极浅杏），图标色 `#F28C5E`
- 导航图标：未选中 `#9E9890`，选中 `#F28C5E`
- 不使用面性、手绘、精美风格图标
- 推荐图标库：Lucide Icons / Phosphor Icons（线性风格统一）

## 六、交互与动效

保持克制，只在必要处加动效。

| 场景 | 动效 | 时长 |
|------|------|------|
| 页面切换 | 简单淡入淡出 | 200ms |
| 列表加载 | 逐条淡入 | 150ms，间隔 30ms |
| 按钮点击 | 透明度 0.7 反馈 | 100ms |
| 下拉刷新 | 系统默认指示器，颜色改为 #F28C5E | — |
| 删除 | 左滑 + 高度收缩消失 | 200ms |

不使用：弹跳、缩放、3D 翻转、数字滚动等花哨动效。

## 七、Web 端响应式

| 断点 | 布局 |
|------|------|
| < 640px | 单列，与移动端一致 |
| 640-1024px | 内容区最大宽度 480px 居中 |
| > 1024px | 左侧窄导航栏 (200px) + 右侧内容居中 |

Web 端不需要三栏布局，记账工具信息密度不高，保持简单。

---

## 八、设计原则总结

1. **留白即设计** — 不用装饰填满空间，让内容呼吸
2. **一个主色走天下** — 只用 `#F28C5E` 一个强调色，避免花哨
3. **线性优于面性** — 图标、图表都用细线条，保持轻盈
4. **去阴影化** — 用极淡描边代替阴影区分层级
5. **数字是主角** — 金额用大号等宽字体，其余元素退后
6. **克制动效** — 只做反馈性动效，不做表演性动效
